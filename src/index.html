Installation

Install mango-cli once from npm and use it everywhere.

npm install -g mango-cli

It will take a while. Go get a coffee.

Usage

  • mango init scaffolding and initialization
  • mango install dependency installation
  • mango build production build
  • mango dev development mode with watch and stuff.

Feel free to use mango [command] -h for detailed instructions.

Managing project dependencies

mango install [packages...]

Installs packages from NPM and stores them in ./node_modules directory, from where you can require them (thanks to Browserify). Maintain current dependencies in mango.json config file under the dependencies section.

Configuration

Your mango.json should look like this.

{
	"src_folder": "src",
	"dist_folder": "dist",

	"styles": [
		"src/styles/screen.styl"
	],
	"scripts": [
		"src/scripts/index.js"
	],
	"images": [
		"src/images/**/*.{jpg,png,svg}"
	],
	"templates": [
		"src/**/*.{jade,html}"
	],
	"dependencies": [
		"jquery"
	]
}
